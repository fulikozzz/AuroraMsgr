# AuroraMsgr
Консольный мессенджер (клиент+сервер) на Go. Курсовая работа по Операционным системам

## Возможности

- Регистрация и вход по логину/паролю
- Личные сообщения между пользователями в реальном времени
- История переписки с отметками времени
- Список диалогов и онлайн-пользователей
- Терминальный интерфейс (TUI) на базе tview
- Шифрование трафика (TLS 1.3)
- Логирование в файл (входы, выходы, сообщения)
- Хранение данных в SQLite
- Работа в локальной сети и через интернет
- Сборка под Linux и Windows

## Стек

| Компонент | Технология |
|-----------|-----------|
| Язык | Go 1.21+ |
| Сеть | TCP + TLS 1.3 |
| БД | SQLite (modernc.org/sqlite) |
| TUI | tview + tcell |
| Логи | log/slog |

## Архитектура
cmd/
server/   — точка входа сервера
client/   — точка входа клиента
internal/
auth/     — аутентификация пользователей
chat/     — Hub для маршрутизации сообщений
logger/   — логирование в файл и консоль
protocol/ — JSON протокол обмена пакетами
storage/  — слой работы с SQLite
configs/
cert.pem  — TLS сертификат
key.pem   — приватный ключ
gencert.go — генератор сертификата
logs/         — файлы логов
docs/         — документация и дневник разработчика

## Быстрый старт

### Генерация TLS сертификата
```bash
go run configs/gencert.go
```

### Запуск сервера
```bash
# Локально
go run ./cmd/server

# Свой порт и БД
go run ./cmd/server -addr 0.0.0.0:9090 -db /var/data/aurora.db

# Без TLS (для отладки)
go run ./cmd/server -tls=false
```

### Запуск клиента
```bash
# Локально
go run ./cmd/client

# Подключение к удалённому серверу
go run ./cmd/client -server 192.168.1.100:8080

# Без TLS
go run ./cmd/client -tls=false
```

### Сборка бинарников
```bash
# Текущая платформа
make build

# Linux
make build-linux

# Windows
make build-windows

# Все платформы
make build-all
```

## Протокол

Клиент и сервер общаются через TCP (с TLS) пакетами в формате JSON,
разделёнными символом `\n`:

```json
{"type":"message","from":"alice","to":"bob","payload":"Привет!"}
```

| Тип пакета | Описание |
|------------|---------|
| login | Вход пользователя |
| register | Регистрация |
| success | Успешная операция |
| error | Ошибка |
| message | Личное сообщение |
| system | Системное сообщение |
| history_request | Запрос истории диалога |
| history | Ответ с историей |
| dialogs_request | Запрос списка диалогов |
| dialogs | Ответ со списком диалогов |

## Управление в TUI

| Клавиша | Действие |
|---------|---------|
| F1 | Фокус на список онлайн |
| F2 | Фокус на список диалогов |
| Tab | Вернуться в поле ввода |
| PgUp/PgDn | Скролл сообщений |
| Ctrl+C | Выход |

Формат сообщения: `@имя текст`  
Пример: `@bob Привет!`
